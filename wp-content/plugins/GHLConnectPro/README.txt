== Description ==

**GHL Connect for WooCommerce -Pro** is a plugin that connects the WordPress/WooCommerce with Go High Level CRM.

== Features ==

* Create a contact on GHL when a wordpress user is registered.
* When someone purchases any products from woocommerce:
 * Specific Tag or Tags can be applied to that contact 
 * Specific Workflow or Workflows can be applied to that contact 
 * All the above are added in the ghl contact based on the Trigger Option you set.
* Support all types of products.
* Sync All Wordpress Users to GHL CRM.
* Set Global Tags and this tags get fired when there is no products specific tags. 
* When someone purchases any products from woocommerce Invoice will be created in GHL CRM.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Go to Add New section of Plugins Menu and search for 'GHL Connect for WooCommerce -Pro'.
2. Now you can install the 'GHL Connect for WooCommerce -Pro' Plugin.
3. Activate the plugin .
4. Go to GHL Connect settings page and connect it with your goHighLevel location.


== Frequently Asked Questions ==

= How to connect my ghl account?=

Simply, Go to the GHL Connect settings page and click "Connect Your Location" button and choose your ghl account to connect.

= Is there any prerequisite to use that plugins? =

Yes, you need to install the woocommerce plugin.

= What is Trigger options tab? =
Its an option where you can select your order status based on that after purchesed the contact will be saved in ghl account.
